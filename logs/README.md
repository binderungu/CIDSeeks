# Logs Directory

Folder ini untuk runtime log lokal (mis. `simulation.log`).

Catatan:
- Log bersifat artefak eksekusi lokal.
- Bukan sumber metrik paper.

## Scope evaluasi (penting)
- Audit terbaru di repo ini menilai **Evaluasi 2** (simulator protokol-level).
- **Evaluasi 1** dan **Evaluasi 3** masih belum final/dibuat penuh, sehingga putusan readiness top venue tidak boleh digeneralisasi ke seluruh paper.
- Saat melaporkan status artefak, gunakan framing: `Eval-2 strong candidate`, sementara status final paper tetap bergantung pada kelengkapan Eval-1 + Eval-3.
